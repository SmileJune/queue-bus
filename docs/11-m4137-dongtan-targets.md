## 요약

M4137 일반 노선의 GBIS `routeId`는 `234001622`입니다. M4137(예약) 노선은 `233000461`이며, 일반 M4137과 분리해서 봐야 합니다.

QueueBus의 1차 분석 대상은 출근 시간대 동탄에서 서울역 방향으로 출발하는 차량과, 퇴근 시간대 서울에서 동탄 방향으로 출발하는 차량입니다. GBIS 위치 API에서 `remainSeatCnt`가 정상 제공되므로, 아래 정류장을 대상으로 차량별 잔여좌석 스냅샷을 누적합니다.

| 퇴근 | 16:00~20:30 KST | 서울→동탄 방향 6개 승차 정류장 |

## 수집 실행

`data/gbis-targets.json`에 위 14개 정류장을 반영했습니다.

한 번 확인:

```bash
npm run gbis:collect:once
```

퇴근/출근 피크 시간대 누적:

```bash
npm run gbis:collect
```

추정치 생성:

```bash
npm run gbis:derive
```

수집 현황 요약:

```bash
npm run gbis:summarize
```

## 현재 확인 사항

- `GBIS_SERVICE_KEY`는 버스위치정보, 정류소 조회, 정류소 경유노선 조회에서 정상 동작했습니다.
- `경기도_버스노선 조회` API는 현재 403 응답이 나왔습니다. routeId는 정류소 경유노선 조회로 우회 확인했습니다.
- 2026-05-28 13:44 KST 기준 M4137 일반 노선 위치 API에서 운행 차량 7대와 잔여좌석 값이 확인됐습니다.
- 서울→동탄 방향은 GBIS 노선 정류장 조회로 국가인권위.안중근활동터, 서울시청, 신한은행본점, 서울역버스환승센터, 명동입구, 명동성당 6개 승차 후보 정류장을 확인했습니다.
